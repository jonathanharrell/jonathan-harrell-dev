The humble input field. Or maybe not so humble… This handy cheat sheet will help you quickly reference all the different text and temporal input types available in HTML. The next time you need to collect information from your users, make sure you are using the right input for the job.

## Text Inputs

| Name | Code | Example |
| -------- | ------- | ------- |
| Text input | `<input type="text" />` | <input type="text" placeholder="Enter text" /> |
| Search input | `<input type="search" />` | <input type="search" placeholder="Enter search term" /> |
| Number input | `<input type="number" />` | <input type="search" placeholder="1234" /> |
| Email input | `<input type="email" />` | <input type="email" placeholder="name@email.com" autoComplete="off" /> |
| Password input | `<input type="password" />` | <input type="password" placeholder="••••••••••" autoComplete="off" /> |
| Telephone input | `<input type="tel" />` | <input type="url" placeholder="999-999-9999" /> |
| URL input | `<input type="url" />` | <input type="url" placeholder="https://yoursite.com" /> |

## Temporal Inputs

| Name | Code | Example |
| -------- | ------- | ------- |
| Date input | `<input type="date" />` | <input type="date" /> |
| Time input | `<input type="time" />` | <input type="time" /> |
| Month input (Chrome/Edge only) | `<input type="month" />` | <input type="month" /> |
| Week input (Chrome/Edge only) | `<input type="week" />` | <input type="week" /> |
| Datetime input | `<input type="datetime-local" />` | <input type="datetime-local" /> |
